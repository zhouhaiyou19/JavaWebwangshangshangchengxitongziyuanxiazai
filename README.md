- **商品查询**：用户可以查询商品信息，查看商品详情。
- **用户注册与登录**：用户注册后可以登录系统，进行购物车管理、订单提交等操作。
- **购物车管理**：用户可以将商品添加到购物车，也可以从购物车中删除商品。
- **订单提交**：用户可以提交订单，完成购物。

- **商品管理**：管理员可以对商品进行增删改查操作。
- **会员管理**：管理员可以管理会员信息。
